There are several ways to get GCC-XML:
  • Download the pre-compiled binary installation for Windows
  • Download the pre-pathed GCC 3.0 source and build it. This is the recommended approach for UNIX users. You may still need to get part of GCC-XML by CVS, however. See Running page for details.
  • Check out the GCC-XML patch from our CVS repository and apply it to a separately obtained GCC 3.0 source. See the page on Patching GCC for instructions.

To use CVS, you must have it installed on your system. You may wish to use the Cygwin tools on Windows platforms, or WinCVS which provides a very nice GUI to CVS.

The source is accessed by checking out a read-only version of the GCC-XML source code. (Only developers have write-access to the repository. Contact Kitware at kitware@kitware.com if you require write access.) Use the following commands.

cvs -d :pserver:anonymous@public.kitware.com:/insight/cvsroot login
(respond with password insight)

Follow this command by checking out the source code (this does not include GCC's source):
cvs -d :pserver:anonymous@public.kitware.com:/insight/cvsroot co GCC_XML
